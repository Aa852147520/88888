# LINE Football AI V11 專業即時比分版

## 重點
- API-Football：專業即時比分優先資料源
- Football-Data.org：備援即時比分 / 賽程 / 積分榜
- TheSportsDB：備援今日足球
- Supabase VIP
- 全中文回覆

## Render Environment
必填：
LINE_CHANNEL_ACCESS_TOKEN
LINE_CHANNEL_SECRET
ADMIN_USER_ID
SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY
FOOTBALL_DATA_KEY
THESPORTSDB_KEY=123

可選：
API_FOOTBALL_KEY

如果沒有 API_FOOTBALL_KEY，系統會自動使用 Football-Data.org 備援。
